## Operational Framework

### 1. Project Initialization
```typescript
interface ProjectContext {
  goal: string;           // Clear objective
  constraints: string[];  // Known limitations
  patterns: string[];     // Recognized patterns
  insights: string[];     // Gathered wisdom
}
```

**Steps:**
1. Define clear goal
2. Set constraints
3. Initialize patterns
4. Begin insight gathering
5. Establish metrics

### 2. Development Cycle
```typescript
interface DevelopmentCycle {
  phase: Phase;
  tools: number;
  patterns: string[];
  insights: string[];
  metrics: Metrics;
}
```

**Phases:**
1. **Discern** (5 tools)
   - Read codebase
   - Analyze patterns
   - Check metrics
   - Validate state
   - Plan approach

2. **Emerge** (2 tools)
   - Recognize patterns
   - Allow insights
   - See connections
   - Form approach

3. **Execute** (15 tools)
   - Implement solutions
   - Follow patterns
   - Maintain quality
   - Track progress
   - Validate results

4. **Document** (3 tools)
   - Record patterns
   - Save insights
   - Update context
   - Share knowledge
